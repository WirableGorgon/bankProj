# Bank Portfolio Project

Учебный банковский клиент на `redom` + `Navigo` с локальным API на `Express`.

## Запуск

1. Установите зависимости фронтенда:
   `npm install`
2. Установите зависимости бэкенда:
   `cd SERVER && npm install`
3. Запустите API в отдельном терминале:
   `cd SERVER && npm start`
4. Запустите фронтенд в корне проекта:
   `npm run dev`

По умолчанию:

- фронтенд доступен на `http://localhost:8080`
- API доступен на `http://localhost:3000`

## Демо-доступ

- логин: `developer`
- пароль: `skillbox`

## Основные сценарии

- вход в приложение
- просмотр списка счетов и создание нового счета
- просмотр деталей счета и перевод средств
- история баланса по счету
- обмен валют
- карта банкоматов

## Примечания

- Все ответы API возвращаются в JSON-формате `{ payload, error }`.
- Для защищенных маршрутов используется заголовок `Authorization: Basic <token>`.
- В проекте установлен Cypress, но актуальная проверка сейчас ориентирована на ручной smoke-test ключевых сценариев.
- Бэкенд можно быстро проверить командой `cd SERVER && npm test`.
